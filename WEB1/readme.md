TAG 
<>string</>다음과 같은 방식으로 표현을 할 수 있다.
ex) <strong>지환</strong>, <u>지환</u> 

통계적으로 가장 많이 쓰이는 TAG들이 있다. 
1. br : 한칸 밑으로 내린다.

2. p : 단락을 나타내는 것.
css라고 해서 <p style="margin-top:40px;" style을 붙여서 좀 더 다양한 시각적 표현을 추가 할 수 있다.

3.img : 이미지를 추가할 수 있다.
ex) <img src="pwd" width="100%">

4.li : 리스트화 시킨다. 

5.ul(Unorderd list) : li의 부모태그로 리스트를 그룹화 할 때 사용한다.

6.ol(orderd list) : odrering을 해준다.

7.title : 제목 정하기.

8. a : 링크 달기.
ex) <a href="https://www.w3.org/" target="_blank" title="W3C main homepage">W3C</a>
